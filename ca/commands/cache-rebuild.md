# cache:rebuild
Reconstruir i esborrar la memòria cau del lloc web.

**Usage:**
```
drupal cache:rebuild [arguments]
cr
cc
```

## Available arguments
Argument | Details
---------|-------------
cache | Esborrar una memòria cau específica.

## Examples
* Reconstruir tota la memòria cau.
```
drupal cr all
```
* Reconstruir la memòria cau "discovery"
```
drupal cr discovery
```
