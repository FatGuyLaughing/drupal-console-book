# theme:uninstall
Desinstal·lar els temes de l'aplicació

**Usage:**
```
drupal theme:uninstall [arguments]
thu
```

## Available arguments
Argument | Details
---------|-------------
theme | Els temes a desinstal·lar han d'estar separats per comes

## Examples
* Uninstall theme specifying the name
```
drupal theme:uninstall  mytheme
```
