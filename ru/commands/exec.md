# exec
Выполнить внешнюю команду.

**Использование:**
```
drupal exec [arguments] [options]
```

## Доступные параметры
Команда | Детали
-------|-------------
--working-directory | Текущая рабочая директория.

## Доступные аргументы
Аргумент | Детали
---------|-------------
bin | Имя выполняемой команды.
