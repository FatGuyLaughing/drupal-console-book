# yaml:update:value
Обновить значение для заданного ключа в YAML файле.

**Использование:**
```
drupal yaml:update:value [arguments]
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
yaml-file | Путь к YAML файлу для обновления
yaml-key | YAML ключ для обновления
yaml-value | YAML значение для обновления
