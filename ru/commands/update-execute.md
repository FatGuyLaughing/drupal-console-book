# update:execute
Выполнить указанный update_N хук в модуле или выполнить все

**Использование:**
```
drupal update:execute [arguments]
upex
updb
```

## Доступные аргументы
Аргумент | Детали
---------|-------------
module | Имя модуля.
update-n | Указанный update_N хук будет выполнен

## Примеры
* Выполнить все обновления
```
drupal update:execute
```
