# migrate:rollback
Откатывает изменения для одной или множества миграций

**Использование:**
```
drupal migrate:rollback [arguments] [options]
mir
```

## Доступные параметры
Команда | Детали
-------|-------------
--source-base_path | commands.migrate.setup.options.source-base-path

## Доступные аргументы
Аргумент | Детали
---------|-------------
migration-ids | Id миграции(й)
